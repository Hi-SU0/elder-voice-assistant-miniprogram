# 老年人语音助手小程序 (Elder Voice Assistant Mini Program)

这是一个专为老年人设计的语音助手微信小程序，旨在通过简单的语音交互，帮助老年人跨越“数字鸿沟”，更方便地使用智能手机获取日常信息、进行健康管理与生活辅助。

## 🌟 核心功能特性

- **👵 适老化设计**：大字体、高对比度界面，操作逻辑简单直观，消除使用门槛。
- **🗣️ 智能语音交互**：支持方言和普通话识别，提供语音播报功能，无需频繁看屏幕。
- **⏰ 日常生活提醒**：吃药、天气预报、日程安排等一句话语音设置提醒。
- **💊 健康常识问答**：内置健康养生百科知识库，解答老年人关心的健康问题。
- **🆘 一键紧急联系**：在紧急情况下，可快速拨打家人电话或发送求助信息。

## 🛠️ 技术栈与架构

- **前端框架**：微信小程序原生开发
- **后端服务**：微信云开发 (WeChat Cloud Development)
  - **云数据库**：存储用户信息、提醒事项、健康问答数据。
  - **云存储**：存放音频文件、图片等资源。
  - **云函数**：处理语音识别、定时任务提醒等复杂业务逻辑。

## 🚀 快速开始

1. **获取代码**：
   ```bash
   git clone https://github.com/Hi-SU0/elder-voice-assistant-miniprogram.git
   ```
2. **导入项目**：打开微信开发者工具，导入该项目目录。
3. **环境配置**：
   - 确保你已经开通了微信云开发环境。
   - 在 `app.js` 或配置中修改云开发环境 ID（`env`）为你自己的环境 ID。
4. **部署云资源**：右键点击 `cloudfunctions` 目录，部署所需的云函数。
5. **编译运行**：点击编译，在模拟器或真机上预览效果。
